# Trilha Tech - Lógica e Pseudocódigo

Uma trilha interativa para o aprendizado de algoritmos, lógica de programação e pseudocódigo (Portugol). 

## Conteúdo Programático
- O que é Algoritmos?
- O que é Fluxograma?
- O que é Pseudocódigo?
- Variáveis e Tipos de Dados
- Entrada e Saída de Dados
- Operadores Aritméticos
- Operadores Relacionais
- Tabela Verdade
- Operadores Lógicos (e, ou, não)
- Estruturas Condicionais (se...senão)
- Laboratório Prático (Desafio Escolar)

## Tecnologias Utilizadas
- HTML5
- Vanilla Javascript
- TailwindCSS (via CDN)
- FontAwesome Icons
- Google Fonts (Inter, Fira Code)
